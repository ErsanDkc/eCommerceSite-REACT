# E-Commerce Site:
## React E-Commerce App Features
-Grid and Slide view of products <br/>  
-Dynamic product detail view specific to GridView and SlideView <br/>
-Dynamic navbar specific to user role (Visitor, User, Admin) <br/>
-Adding products, deleting products, updating products, viewing orders, deleting orders via Admin Panel <br/>
-Responsive Design 

## Tools
-ReactJS <br/>
-React Query <br/>
-Chakra-Ui <br/>
-Ant Design <br/>
-React Bootstrap <br/>
-Context API <br/>
-Formik
